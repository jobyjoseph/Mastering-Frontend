In order to use numerical data, there are 6 data types in Kotlin.

```kotlin
var a : Byte = 5;
var b : Short = 50;
var c : Int = 500;
var d : Long = 5000;
var e : Float = 50000.10f;
var f : Double = 500000.00;
```
